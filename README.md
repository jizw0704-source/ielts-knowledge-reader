# IELTS Knowledge Reader

## 项目简介
IELTS Knowledge Reader 是一个手机端 H5 雅思知识阅读器 MVP。
它不是雅思真题售卖平台，而是一个围绕原创 IELTS-style 英文文章、点词释义和生词本设计的阅读学习工具。

## 当前功能
- 今日文章首页
- 阅读页
- 点词释义弹窗
- 加入生词本并写入 `localStorage`
- 生词本列表、删除和熟悉度管理
- 文章库和标签筛选
- 固定底部导航

- 文章单词数：根据正文 `content` 自动统计英文单词数
